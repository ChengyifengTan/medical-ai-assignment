# 项目名称
Medical AI Course - Hands-on Practice Notebook Generator

# 项目目标
本项目旨在为《医学人工智能》课程生成 **教学友好的 Jupyter Notebook 随堂实践脚本**，主要面向**无编程背景或编程基础较弱的医学生**。

Claude 的核心任务是：
> 生成“可以直接用于课堂讲解 + 学生动手实践”的 Notebook 教学材料

---

# 🎯 教学对象与设计原则

## 学生背景
- 医学生
- 编程基础弱或接近零基础
- 数学基础有限（避免复杂推导）
- 更容易理解**直觉解释 + 应用场景**

## 教学原则
1. **通俗易懂 > 数学严谨**
2. **案例驱动 > 抽象理论**
3. **逐步引导 > 一步到位**
4. **可运行 > 复杂最优**

---

# 📦 Notebook 输出格式规范（必须严格遵守）

每个 Notebook 必须包含以下结构：

## 1. 标题与任务介绍（Markdown Cell）
- 项目名称（贴近医学场景）
- 本节课学习目标（bullet points）
- 本次任务说明

示例风格：
- “我们将用AI方法帮助医生分析……”

---

## 2. 医学背景引入（Markdown Cell）
必须包含：
- 真实或拟真的医学应用场景
- 为什么这个问题重要（临床意义）
- 数据是什么（例如：患者指标、影像、心电信号等）

---

## 3. 方法原理讲解（Markdown Cell）
要求：
- 不使用复杂公式
- 使用类比、直觉解释
- 可以用“如果你是医生，你会怎么做”引导理解

示例：
- 聚类 → “把相似的病人分成一组”
- 分类 → “判断病人属于哪种疾病”
- 回归 → “预测一个具体数值（比如血压）”

---

## 4. 数据准备（Code + Markdown）
要求：
- 使用简单、干净的数据集（推荐 sklearn / 自造数据）
- 数据字段必须有“医学意义命名”

示例：
```python
# 示例字段
age = ...
blood_pressure = ...
cholesterol = ...
```

---

## 5. 核心代码讲解（重点部分）

### 🔴 强制要求：

#### （1）代码必须可运行

#### （2）必须用注释解释关键代码

#### （3）关键位置必须加入 TODO 及 提示

示例：

```python
# TODO: 使用 StandardScaler 对数据进行标准化
# 提示：调用 sklearn.preprocessing 中的 StandardScaler
scaler = StandardScaler()
X_scaled = scaler.fit_transform(X)
```

---

## 6. 模型构建（Code + Markdown）

要求：

* 使用 sklearn / PyTorch（仅限基础用法）
* 不允许复杂工程代码
* 一步一步拆解

---

## 7. 结果分析（Markdown）

必须包含：

* 模型输出结果解释（不是只打印）
* 用“医生视角”解释结果

示例：

* “模型将患者分为两类，可能对应高风险/低风险人群”

---

## 8. 小练习

每个 Notebook 必须包含：

### ✔ 填空题（基于 TODO）

### ✔ 思考题/扩展练习（开放问题）

示例：

* “如果增加一个指标（BMI），结果会有什么变化？”
* “你觉得模型可能在哪些情况下出错？”

---

## 9. 总结（Markdown）

* 本节课学了什么
* 和医学的关系

---

# ⏱️ 难度与时间控制

每个 Notebook 必须满足：

* ⏰ 讲解 + 实践时间 ≈ 30分钟
* 💻 代码量：50 ~ 120 行
* 🧠 概念数量：≤ 3 个核心点
* 🚫 不允许：

  * 复杂数学推导
  * 多模型对比
  * 工程化代码（如类封装）

# 🧪 各周主题任务要求

## Week 3：数据预处理 + 聚类

任务示例：

* 病人分群（高风险/低风险）
* 使用 KMeans

重点：

* 标准化
* 聚类概念

---

## Week 4：回归

任务示例：

* 预测糖尿病疾病进展

模型：

* Linear Regression

重点：

* 输入 → 输出的连续关系

---

## Week 5：分类

任务示例：

* 疾病分类

模型：

* Logistic Regression / Decision Tree

重点：

* 分类 vs 回归 vs 聚类 区别

---

## Week 6：神经网络

任务示例：

* 疾病分类

模型：

* PyTorch MLP（2层以内）

重点：

* “像大脑一样学习”的直觉

---

## Week 8：卷积神经网络（CNN）

任务示例：

* 医学影像分类（简化版）

模型：

* 简单 CNN（2-3 层）

重点：

* 图像特征提取

---

## Week 9：时序模型

任务示例：

* 心电信号 / 时间序列预测

模型：

* RNN / LSTM（极简版）

重点：

* “过去影响未来”
