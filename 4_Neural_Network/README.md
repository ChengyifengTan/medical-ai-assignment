## 📚 本节课内容

### 核心目标
1. 理解神经网络的基本原理和工作机制
2. 学习用 PyTorch 构建和训练简单的神经网络
3. 理解回归任务和分类任务在神经网络中的区别
4. 理解神经网络训练的基本流程

### 技术要点
- **神经网络结构**：输入层、隐藏层、输出层
- **激活函数**：ReLU、Sigmoid
- **损失函数**：MSE（回归）、BCE（分类）
- **优化器**：Adam
- **训练流程**：前向传播、损失计算、反向传播、参数更新

### 数据集
- 基于第四周的**糖尿病数据集**（442名患者）
- **回归任务**：预测糖尿病病情进展数值（0-346）
- **分类任务**：判断患者是高风险（>150）还是低风险（≤150）

### 3. 医学场景化
- 所有概念都用医学类比解释
- 强调神经网络在医学中的实际应用
- 预测病情 vs 风险分层的临床意义

### 4. 可视化丰富
- 训练过程可视化（损失、准确率变化）
- 预测结果可视化（散点图、概率分布、混淆矩阵）

## 🛠️ 技术栈

### Python库
- **PyTorch**：深度学习框架，用于构建和训练神经网络
- **NumPy**：数值计算
- **Pandas**：数据处理
- **Matplotlib**：数据可视化
- **Scikit-learn**：机器学习工具（数据加载、评估指标）

### 网络结构
```
回归网络：
输入层(10) → 隐藏层1(16, ReLU) → 隐藏层2(8, ReLU) → 输出层(1, 无激活)

分类网络：
输入层(10) → 隐藏层1(16, ReLU) → 隐藏层2(8, ReLU) → 输出层(1, Sigmoid)
```

### 训练配置
- **优化器**：Adam (学习率=0.001)
- **训练轮数**：100 epochs
- **批量大小**：32

## ⚠️ 注意事项

### 常见问题
1. **PyTorch安装**：需要确保Python版本兼容（推荐3.8-3.11）
2. **训练时间**：简单网络训练较快，约1-2分钟
3. **随机性**：设置随机种子以保证结果可复现
4. **过拟合**：如果训练集远好于测试集，可能存在过拟合

### 调试技巧
1. 先运行参考答案，确保环境正常
2. 逐步检查学生填写的代码
3. 观察训练过程的loss变化曲线
4. 检查数据形状和类型是否正确
